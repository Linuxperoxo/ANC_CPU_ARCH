# O que é esse projeto?

Este projeto foi criado com o intuito de estudo e compreensão dos princípios de funcionamento de um processador. O objetivo é simular e entender suas operações e seu funcionamento. Contudo, decidi que iria expandir esse projeto, 
decidi então, fazer um compilador exclusivo para o ANC, a arquitetura que fiz do zero. O NANC vai ser o compilador para a linguagem que estou fazendo somente para esse microprocessador, o "CANEBLY Lang". Canebly Lang é 
nossa linguagem assembly para o SLENK, a ideia que eu tive para ter um melhor controle sobre o arquivo binário é simplesmente escrever 0 e 1 nele, como um arquivo de texto, mais pra frente vou remover isso e adicionar a criação
de um arquivo binário mais elaborado. Escrever 0 e 1 em um arquivo se torna uma tarefa extremamente legal porém com uma dificuldade imensa de leitura, logo, o Canebly vai ajudar muito a criar software para nosso microprocessador.

Espero que você goste desse projeto :). 

Agradeço qualquer feedback ou contribuição!
